Covid Analysis

As a start, I made a 3-steps project going over the ETL and visualization of Covid-19 data.
The data was extracted from the source, imported to a SQL database to be transformed, then I created relevant views and exported them to finally load them on Tableau to create a Dashboard.

Exploratory Analysis using SQL

Exploratory Analysis for Covid data using SQL

The data was extracted from the source and saved .CSV to be then uploaded to a local MS SQL Server. After understanding how the data was structured, it was time to aggregate key metrics such as infection and death rates and analyse it through relevant dimensions, such as Date, Continent, Countries and Income. The key insights derived from this were saved as views to be used later.
