My Projects

As a Business and Data Analyst, I developed skills in multiple tools
Here are a few projects I developed to showcase my expertise.

Covid Analysis

As a start, I made a 3-steps project going over the ETL and visualization of Covid-19 data.
The data was extracted from the source, imported to a SQL database to be transformed, then I created relevant views and exported them to finally load them on Tableau to create a Dashboard.

Exploratory Analysis using SQL

1 - Exploratory Analysis for Covid data using SQL

The data was extracted from the source and saved .CSV to be then uploaded to a local MS SQL Server.
After understanding how the data was structured, it was time to aggregate key metrics such as infection and death rates and analyse it through relevant dimensions, such as Date, Continent, Countries and Income.

  • view code on Github

2 - Data Views using SQL

From the data extracted and transformed on the first step, 6 different views were planned and then created using SQL, so they could be saved on MSSQL Server as a backup.
This views were then extracted to an Excel Sheet, ordered and checked for format consistency, so it could be ready to be used on a Tableau Dashboard.

  • view code on Github

3 - Tableau Dashboard

Lastly, a visual data analysis using Tableau Public was done using the 6 views as data input for different types of charts.
The data was displayed by using the charts that better showcased the insights, such as using a map to show the infection rate per Country and using a time series to show a predictive analysis of the expected infection rate of the BRICS countries until the end of 2024.
